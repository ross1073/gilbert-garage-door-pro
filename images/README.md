# Image Naming Guide — Garage Door Repair in Gilbert, AZ

## Naming Convention

`{page-slug}_{descriptor}_{number}.jpg`

- **First part** = which page it goes on (see slugs below)
- **Middle part** = what it shows (hero, showcase, before-after, project, team)
- **Last part** = number if you have multiples

## Page Slugs

- `home` — garage door repair in Gilbert, AZ
- `repair-garage-door-openers` — Repair Garage Door Openers
- `garage-door-openers-repair` — Garage Door Openers Repair
- `garage-door-opening-repair` — Garage Door Opening Repair
- `repair-garage-door-opener` — Repair Garage Door Opener
- `repair-a-garage-door-opener` — Repair A Garage Door Opener
- `close-to-me` — Garage Door Repair Close To Me
- `garage-door-garage-door-repair` — Garage Door Garage Door Repair
- `repair-for-garage-door` — Repair For Garage Door

## Examples

```
home_hero_1.jpg              — Hero banner for homepage
home_showcase_1.jpg          — Showcase image on homepage
repair-garage-door-openers_project_1.jpg     — Project photo for service page
repair-garage-door-openers_before-after_1.jpg — Before/after for service page
gallery_project_1.jpg        — Portfolio gallery image
about_team_1.jpg             — Team photo for about page
```

## Image Requirements

- **Any size is fine** — Claude will optimize during the site build
- **Format:** .jpg for photos, .png only if you need transparency
- **Quality:** Use your best photos — real job site photos perform much better than stock
- **Quantity:** Aim for at least 1 hero + 3-5 project photos
